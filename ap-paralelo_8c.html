This is the main function. It initializes the image handler, creates the pipelines and threads, and then waits for the threads to finish.

Parameters
[in]argc- number of arguments
[in]argv- array of arguments
[out]0if the program runs successfully
Note
Side Effects:
  • Initializes the image handler.
  • Creates the pipelines and threads.
  • Waits for the threads to finish.
  • Prints the time it took to run the program.
  • Prints the number of pipelines and the directory where the output files will be placed.
  • Prints the directory where the input files are located.
  • Prints the number of images to be processed.
  • Prints the name of the images to be processed.
  • Prints the name of the stages performed by the threads.
  • Prints the size of the images after each stage.
  • Prints the name of the output files.
